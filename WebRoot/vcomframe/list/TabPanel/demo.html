    一、功能简介

    标签面板可以固定的位置，显示多种信息，提高了页面空间使用率。
        

    二、使用说明

     1、在页面中引入tabpanel.js文件
    例如：< script="javascript" src="vcomframe/list/TabPanel/tabpanel.js">< /script>
2、然后在页面上需要使用TabPanel的位置编写代码
        < div class="tab-pane" style="height:300px;width:600px">
            < div class='tab-page'>
                < div class='tab'>分页一
                分页一显示内容
            < /div>
            < div class='tab-page'>
                < div class='tab'>分页二< /div>
                分页二显示内容
            < /div>
         < /div>
